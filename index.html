Game Dev Tycoon Calculator

Plan each release around the scoring formulas used by the game and keep your next target score under control.

The formulas and data in this project come from community research around the original calculator. This cleanup keeps the calculator working as a standalone static site and stores your recent games locally in the browser.

How To Use It

Pick the platform, topic, genres, audience, game size and year, then enter your tech, design and bug totals. The calculator estimates game score, review score, slider targets and the score you should aim to beat next.

Use Next Game to build a local history. Save and Load persist that history in your browser using local storage.

Background

This repository is a preserved and cleaned up version of an older fan-made calculator that was originally distributed through archived static pages.

For broader strategy details, see the Game Dev Tycoon success guide .
